# Template Helper
Chrome extension. In MI atlas process of Email Templates testing is quite time consuming. 2nd line has to login as user who was requesting the template on testing environment, then create Case with preferred data and after saving - create Email response. This is about 12 (!) mouse clicks.  This extensions tries to solve this problem, by reducing that to (minimum) 2 mouse clicks.
